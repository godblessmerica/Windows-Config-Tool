# Windows Config Tool
A lightweight batch utility for users who want to customaize windows without having to navagate complex settings themselfs. This tool is specificly for users with unactivated windows where personalization settings are locked. Its only for Windows 11.
## Features
* 🌓 **Theme Change** Force Enable Dark Mode or Light Mode.
* ✂️ **Watermark Removal:** Disables the "Activate Windows" watermark (Requires Reboot).
* 🔐 **MAS Interated** Has built in intergration with MAS to permanently activate Windows.
* 🎨 **UI Refresh** Automaticly restarts Windows Explorer to apply theme changes

## How To Use
1. Dowload The .bat file
2. Right-click the file and run it as administrator (this is required so it can edit the registry files and run MAS)
3. Choose an option from the menu.
